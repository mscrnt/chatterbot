{# Dashboard-wide SSE event dispatcher. Opens one EventSource against /events/stream. On each `event: ` message, dispatches a DOM event of the same name on `body` so HTMX panels can listen via `hx-trigger=" from:body"`. Replaces the old per-panel `hx-trigger="every Xs"` polls. Idle cost is one heartbeat ping every ~22 s. State-change cost is one tiny SSE event + the panel's own HTMX refetch. The server-side watermark poll runs every 1.5 s but only emits when something actually moved. The driver auto-reconnects with capped exponential backoff on error. #}
