{# Live-chat SSE driver. Replaces the old `hx-trigger="every 2s"` poll with a single EventSource connection per page open. The server pushes only NEW message rows, which we prepend to any element marked `data-live-feed`, then trim past `data-live-limit`. Initial render is server-side (HTMX one-shot for the sidebar, the page template itself for /live). The SSE stream takes over from there. Idle cost: ~0 bytes/s once the heartbeat lull settles in.
