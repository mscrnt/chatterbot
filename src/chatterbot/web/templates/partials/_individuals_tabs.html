{# Vanilla JS driver for the Individuals panel (Talking-to-you / Active-but-not-engaged tabs + show-more queue expansion). Bypassing Alpine here because the panel re-renders on every SSE-triggered body swap, and Alpine's re-hydration on this specific subtree was unreliable — both panes would go blank after a few swaps. Delegated body-level click handlers run regardless of how often the DOM gets replaced; the apply() pass after each swap re-syncs visibility from localStorage so the streamer's choice survives. localStorage keys (unchanged from the Alpine version): chatterbot.insightIndividualsTab — 'mentions' | 'lurkers' chatterbot.mentionsExpanded — '1' | '0' chatterbot.lurkersExpanded — '1' | '0' #}
